## üî• Practical Descriptions

### **1. Introduction to Predictive Analytics**

* Understanding ML workflow and predictive modeling
* Demonstrated Random Forest classification
* Output includes accuracy & classification report

### **2. Data Cleaning & Handling Missing Values**

* Imputed missing numeric & categorical values
* Used SimpleImputer (Mean/Mode) & IterativeImputer
* Ensured cleaned & complete dataset

### **3. Outlier Detection & Feature Engineering**

* Outliers detected using IQR method
* Applied capping (winsorization), Power Transform
* Feature scaling for model readiness

### **4. Feature Selection & Train-Test Split**

* Feature reduction using SelectKBest + Correlation
* Dataset split using train-test split technique
* Regression modeling with R¬≤ score

### **5. Simple Linear Regression**

* Built linear regression using `statsmodels OLS`
* Plotted **Q-Q Plot** and **Residual Plot**
* Verified regression assumptions

### **6. Evaluation Metrics & Residual Analysis**

* Calculated **MSE, RMSE, MAE, R¬≤** metrics
* Visualized residual distribution
* Assessed model error spread & bias

### **7. Decision Tree Classifier**

* Model trained on Iris dataset
* Accuracy, Confusion Matrix, Report generated
* Visualized decision tree structure

### **8. Time Series Forecasting (ARIMA)**

* ARIMA(1,1,1) forecasting model implementation
* Visual forecast vs actual comparison
* Useful for trend-based prediction tasks

### **9. Model Selection & Hyperparameter Tuning**

* Compared multiple ML models using Cross Validation
* Applied GridSearchCV on Random Forest
* Created model accuracy comparison chart
